# Shopify Embedded App Playground

A playground for learning and experimenting with Shopify embedded app development using the latest Shopify app architecture.

This repository serves as my personal learning space to understand how modern Shopify embedded apps work under the hood—from authentication and App Bridge to the Admin API and webhooks.

## 🎯 Goals

- Learn Shopify embedded app architecture
- Understand OAuth and session management
- Explore App Bridge and Polaris
- Work with the Shopify Admin GraphQL API
- Implement webhooks
- Learn embedded app authentication flow
- Build production-ready patterns for future Shopify apps

## 🛠️ Tech Stack

### Frontend

- React Router
- Shopify App Bridge
- Shopify Polaris
- Vite

### Backend

- Shopify App React Router
- Node.js
- Prisma (default template)
- Shopify Admin GraphQL API

> Future versions of this playground may also explore an Express + Supabase architecture.

## 📚 Topics Covered

- Shopify OAuth
- Access Tokens
- Session Tokens (JWT)
- Embedded App Authentication
- App Bridge
- Shopify Admin API
- GraphQL
- Webhooks
- Route Loaders & Actions
- Session Storage
- Embedded Navigation
- App Extensions

## 📁 Project Purpose

This repository is **not** intended to be a production application.

It is a collection of experiments, notes, code samples, and implementations created while learning Shopify embedded app development. Code may change frequently as new concepts are explored.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/shopify-embedded-app-playground.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
